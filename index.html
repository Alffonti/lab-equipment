Different scenarios

Scenario 1

The container direction is set to row, so the horizaontal axis is the main axis. The center value is used for the main axis (in this case ↔️). The end value is used for the cross axis (in this case ↕️).

Scenario 2

The container direction is set to column, so the vertical axis is the main axis. The space-between value is used for the main axis (in this case ↕️). The center value is used for the cross axis (in this case ↔️).

🥽
🧪
🧫

Scenario 3

No container direction is set. So, it takes the default value: row. So the vertical axis is the main axis. The space-evenly value is used for the main axis (in this case ↔️). The start value is used for the cross axis (in this case ↕️).
